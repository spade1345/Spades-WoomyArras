# Arras: Dungeons

This is a spinoff of my standard Spades-WoomyArras that is built specifically for one mode: Dungeons.
Dungeons is a variation on Boss Rush that replaces bosses with a large amount of various boss shards, sentries, and Army units.


## Playing

There is no official online site for Dungeons. To play, download the zip (CODE > DOWNLOAD ZIP), unzip the downloaded folder, and open index.html.

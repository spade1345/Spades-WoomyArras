# Spade's Woomy Arras  

> A restored offline version of woomy arras

## NOTICE

This is a **fan-made offline** version of woomy-arras. For more updated code, visit <https://github.com/DrakoHyena/WoomyArchives>. For Draco Hyena's online woomy-arras, visit <https://woomy.online>.

## How to play

Download the ZIP file (the green CODE button, then Download ZIP)  
Open index.html  

The game can also be accessed on an online website for those who can't (or are too lazy to) download the source code. The list of websites is as follows.

- <https://woomy-arras.runbsd.io>
- <https://spades-woomy-arras.vercel.app>
- <https://woomy-arras.vercel.app>
- <https://woomy-modded.vercel.app>

## What is this

This is a full backup/copy of Drako Hyena's offline woomy-arras.io files made cicra 2018(?)  
This has been modified by me (Spade1345; or call me the Highwayman) to have many tanks readded. Some other changes have been made, mainly to how AI/bot tanks' names are handled.
